# User Journeys

A vision for intuitive music discovery and social sharing.

---

## Tab Structure

| Tab | Icon | Purpose |
|-----|------|---------|
| Discover | compass | Home feed - new music, recommendations, friend activity |
| Search | magnifying glass | Find specific songs, artists, albums |
| Chats | message bubble | DMs, group chats, shared music |
| Profile | person | Stats, taste profile, settings |

---

## Discover Tab (Home)

The primary tab users land on. A personalized feed of music discovery.

### Sections

**Now Playing (Friends)**
- See what friends are currently listening to in real-time
- Tap to preview the song
- Tap friend's avatar to send them a message about it

**Daily Mix**
- Personalized playlist refreshed each day
- Based on listening history + friend activity
- One-tap to play full mix or preview tracks

**New From Artists You Follow**
- Recent releases from artists in your library
- Sorted by release date
- Quick-save to library or share to friend

**Trending Among Friends**
- Songs your friends have been sharing/saving this week
- Shows which friends liked it
- Social proof drives discovery

**For You (AI Recommendations)**
- "Because you liked [song]" carousels
- Genre exploration based on listening patterns
- Discover Weekly style algorithmic picks

**Mood/Activity Picks**
- Morning commute, workout, chill, focus
- Context-aware suggestions (time of day, day of week)

### User Journeys

**Quick Discovery (30 seconds)**
1. Open app, land on Discover
2. See friend is listening to something interesting
3. Tap to preview
4. Swipe up to save or share

**Deep Discovery (5-10 minutes)**
1. Scroll through Daily Mix, preview tracks
2. Save favorites to playlist
3. Check new releases from followed artists
4. Explore "Because you liked X" recommendations
5. Share best find to group chat

## Search Tab

Fast, powerful search with smart suggestions.

### Features

**Smart Search Bar**
- Recent searches shown below
- Trending searches as suggestions
- Search by song, artist, album, or lyrics snippet

**Voice Search**
- Tap mic icon to search by voice
- "Find that song that goes..."

**Shazam Integration**
- "What's playing?" button
- Identify songs playing around you
- Auto-save identified songs to history

**Search Results**
- Segmented: Songs | Artists | Albums | Playlists
- Preview without leaving results
- Quick actions: Save, Share, Add to Queue

**Filters**
- Release year range
- Genre
- Explicit/clean
- Available in your region

### User Journeys

**Quick Find (10 seconds)**
1. Open Search
2. Type artist or song name
3. Tap result to preview
4. Swipe to save or share

**Identify Song**
1. Hear song playing somewhere
2. Open Search, tap Shazam button
3. Song identified
4. Preview, save, or share to friend with "just heard this!"

**Deep Dive Artist**
1. Search artist name
2. View artist page with top songs, albums, about
3. Explore discography
4. Add multiple songs to playlist
5. Follow artist for new release notifications

## Chats Tab

Social hub for music conversations.

### Conversation Types

**DMs**
- 1-on-1 conversations with friends
- Vibestreak counter displayed
- Rich song sharing with inline playback

**Group Chats**
- Create groups around music interests
- Name and customize group
- Shared playlist that all members can add to

### Message Types

**Text**
- Standard messages
- Emoji reactions

**Song Share**
- Full song card with album art
- Inline 30-second preview
- Quick reactions: fire, heart, 100, meh
- One-tap save to playlist

**Playlist Share**
- Share entire playlist
- Recipients can preview tracks inline
- "Add all to library" option

**Voice Notes (Future)**
- Record short audio message
- "Listen to this part!" with timestamp

### Vibestreaks

Gamification to encourage daily music sharing.

**How It Works**
- Both users must interact daily to maintain streak
- Send song, react to song, or message counts
- Streak visible next to friend's name

**Streak Rewards**
- 7 days: Bronze flame icon
- 30 days: Silver flame icon
- 100 days: Gold flame icon
- 365 days: Special animated icon

**Streak Protection**
- One free "streak freeze" per month
- Notification reminders before streak breaks
- "Your streak with [friend] is about to end!"

### User Journeys

**Quick Share (15 seconds)**
1. Find song (from Search or Discover)
2. Tap share button
3. Select friend
4. Sent with one tap

**Music Conversation**
1. Friend sends you a song
2. Tap to preview
3. React with fire emoji
4. Send a song back "this reminds me of..."
5. Discuss and build on shared taste
6. Streak increases

## Profile Tab

Personal stats, taste identity, and settings.

### Sections

**Profile Header**
- Profile photo and display name
- Username/handle
- Bio (optional music-focused tagline)
- Friend count
- Edit profile button

**Listening Stats**
- Time period selector: This week | This month | All time
- Minutes listened
- Songs played
- Top genres pie chart

**Top Artists**
- Grid of top 5-10 artists
- Based on selected time period
- Tap to view artist page

**Top Songs**
- List of most played songs
- Play count shown
- Tap to play or share

**Music Personality**
- AI-generated taste profile
- "You're an Indie Explorer with Electronic tendencies"
- Shareable card for social media
- Updates based on listening changes

**Badges & Achievements**
- Early Adopter (joined in first year)
- Streak Master (100+ day streak)
- Tastemaker (friends often save songs you share)
- Genre Explorer (listen to 10+ genres monthly)
- Completionist (listened to full album)

**Recently Played**
- Chronological listening history
- Quick replay or share

**Settings**
- Spotify connection management
- Notification preferences
- Privacy settings (who can see stats)
- Theme (light/dark/system)
- Account management

### User Journeys

**Check Stats (1 minute)**
1. Open Profile
2. View this week's listening time
3. Check if top artists changed
4. Share interesting stat to story/chat

**Flex Music Taste**
1. Generate Music Personality card
2. Screenshot or share directly
3. Compare with friends
4. Discover where tastes overlap/differ

**Profile Customization**
1. Tap edit profile
2. Update photo and bio
3. Choose featured top song
4. Set privacy preferences

## Cross-Tab Flows

### New User Onboarding
1. Sign up with email/Apple/Google
2. Connect Spotify (required for full features)
3. Brief tutorial highlighting each tab
4. Add friends via contacts or username
5. Land on Discover with personalized recommendations
6. Prompt to send first song share

### Daily Engagement Loop
1. Push notification: "See what [friend] is listening to"
2. Open app to Discover tab
3. Check friend activity, preview interesting songs
4. Send reaction or share song back
5. Check for new releases
6. Maintain streaks with close friends

### Music Discovery to Sharing
1. Discover → Find new song in recommendations
2. Preview and love it
3. Search → Look up more by artist
4. Add favorites to playlist
5. Chats → Share best find to friend/group
6. Profile → See it appear in top songs next week
